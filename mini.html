Title: Secure Cloud Storage with Data Dynamics using Secure Network Coding Techniques

Description:

The aim of this project is to build a secure cloud storage system that allows users to store, update, and manage their data safely on remote servers using secure network coding techniques.

  • In cloud computing, people store data on remote servers instead of their own devices.
  • The challenge is keeping that data safe, correct, and trustworthy, even when the server is not fully reliable.
This project designs secure storage protocols (called DSCS I and DSCS II) using secure network coding so users can:
  • Upload data securely
  • Check data integrity (make sure it is not changed or corrupted)
  • Update data dynamically (insert, delete, or modify files)
  • DSCS I supports dynamic data (insert, delete, modify).
  • DSCS II supports append-only data (adding new data without changing old data).
  • Both are tested to evaluate their performance.

In short: The project is about storing and updating cloud data securely using coding algorithms, so users can trust their outsourced data.
